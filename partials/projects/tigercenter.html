Tiger Center Enrollment System

For this project, I used a Raspberry Pi model B+ to create a media center and emulation station for the collection of videos, music, and ROMs on my home server. It uses a DualShock 4 controller to control the interface, which is actually wrapped as an Xbox controller, and then as a keyboard due to the lack of joystick support in the Raspberry Pi version of XBMC (now Kodi). Everything is up and running under Arch Linux and the Raspberry Pi boots directly into XBMC, with scripts to start and stop the media center to improve speed whenever a game is loaded.
