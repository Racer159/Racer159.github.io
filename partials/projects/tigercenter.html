Tiger Center Enrollment System

Tiger Center was actually a web application that I helped create as part of the Student IT Office. It basically serves as a fresh interface for the Campus Solutions enrollment application that RIT uses. It uses modern web frameworks such as Angular.js and Foundations for the front end, and provides a much more dynamic and fluid experience for its users. I was lucky enough to be a part of this project from start to finish, and was able to really understand the processes that go into designing, developing, deploying, and maintaining enterprise-level web applications.
